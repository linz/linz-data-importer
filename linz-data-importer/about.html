LINZ Data Importer

This plugin allows you to easily discover and import data from Toitū Te Whenua Land Information New Zealand into QGIS.

Explore and connect to LINZ's WMTS and WFS services from within QGIS.

Available data

Data (available through WFS or WMTS) from the agency data portals and services below are available via the LINZ Data Importer.

Open data portals

  • Toitū Te Whenua – Land Information New Zealand (https://data.linz.govt.nz)
  • Manatū Mō Te Taiao – Ministry for the Environment (https://data.mfe.govt.nz)
  • Tatauranga Aotearoa – Statistics New Zealand (https://datafinder.stats.govt.nz)
  • Manaaki Whenua – Landcare Research (https://lris.scinfo.org.nz)
  • Te Ope Kātua o Aotearoa – New Zealand Defence Force (https://geodata.nzdf.mil.nz)

Basemaps services

  • LINZ Basemaps (https://basemaps.linz.govt.nz)

The plugin supports data available via WMTS and WFS services. If you are unsure which service best suits your needs, see the resources featured in the guides and documentation section of the LINZ website.

Get started

The first step in using the LINZ Data Importer is to get an API key. An API key is required from each service you wish to access data from via the plugin.

See the Help tab for details on how to generate an API keys.
